# 开发备注（不跑本地测试）

## 1. 本地开发允许做什么

- 可以在 Android Studio 运行 App、手动点点功能做自测。
- 可以本地构建 `assembleDebug`（若需要）。

## 2. 本地开发禁止做什么（本项目约束）

- 不在本地运行单元测试或仪器测试：
  - 不执行 `./gradlew test*`
  - 不执行 `./gradlew connected*`

所有测试结果以 GitHub CI 为准。

## 3. 常见提交粒度

- 依赖与工程改造：单独 PR
- Room 落地：单独 PR
- UI 功能：按页面或按功能点拆 PR

## 4. 离线约束提醒

- 不添加 `INTERNET` 权限。
- 任何新依赖都要确认不隐式引入网络行为。
